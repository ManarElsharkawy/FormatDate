# FormatDate
This is a simple cpp Format Date Console App

Date Formatter C++ Console App

A simple C++ console application that allows you to work with dates, format them, validate them, and display them in different styles.
This project is designed to help you practice date handling, structures, comparison functions, and formatting logic.

### Features
Convert dates into different formats
  DD/MM/YYYY
  MM-DD-YYYY
  YYY/MM/DD
  Day: Month: Year:


### Technologies Used
C++
Standard Library(string, vector)
